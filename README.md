## Quick Start

### Generate a new day's template:

```bash
./generate.sh <day_number>
```

Example:
```bash
./generate.sh 1  # Creates day01/ directory with template files
```

### Running solutions:

```bash
cd day01
lua day01.lua input.txt
```

## Structure

Each day's folder contains:
- `dayXX.lua` - Solution code
- `input.txt` - Puzzle input (not committed)
- `example.txt` - Example input from problem description
